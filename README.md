# Tubender landing page

A single-page, bespoke prospect site built in Tubender's own brand vocabulary (teal #00ADB5 primary, deep blue #00588b accent, Helvetica, pill buttons at 30px radius) to host the pitch body from fit-analysis.md section 7. It reframes the software layer that sits above Tubender's CNC conduit bender (quoting, take-off, bend-plan, integrations) and offers a 30-minute conversation with Matt Creamer, CRO at Eduba, plus a warm-route option to NLP Logix for build work. Eduba branding is confined to the footer line and the named case study references; everything else reads as if Tubender's own team wrote it. Mobile-first, no JavaScript, no external fonts beyond the system stack Tubender already uses.
